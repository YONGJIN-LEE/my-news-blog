# 오늘의 뉴스 브리핑 - 에디토리얼 가이드라인

## 1. 사이트 소개

**사이트명:** 오늘의 뉴스 브리핑  
**목적:** 최신 뉴스를 빠르고 정확하게 전달  
**대상 독자:** 한국어 사용자, 일반 대중  
**콘텐츠 유형:** 뉴스 브리핑, 이슈 분석

---

## 2. 콘텐츠 구조

모든 기사는 다음 구조를 따릅니다:

### 2.1 프론트매터 (Front Matter)

```yaml
---
title: "헤드라인 제목"
date: YYYY-MM-DDTHH:MM:SS+09:00
slug: "YYYY-MM-DD-키워드"
category: ""
tags: ["관련태그1", "관련태그2", ...]
thumbnail: "이미지URL"
description: "헤드라인 제목과 동일"
draft: false
---
```

**작성 규칙:**
- `title`: 핵심 내용을 담은 흥미로운 헤드라인 (50자 내외)
- `date`: 한국 표준시(KST, +09:00) 기준
- `slug`: 날짜-핵심키워드 형식 (영문/한글 혼용 가능)
- `tags`: 10-20개의 관련 키워드 (검색 최적화용)

### 2.2 본문 구조

| 섹션 | 설명 | 분량 |
|------|------|------|
| **핵심 요약** | 불릿 포인트로 5개 내외의 핵심 포인트 | 5줄 |
| **사건/이슈 개요** | 전체 맥락을 설명하는 도입부 | 1-2문단 |
| **상세 내용** | 소제목으로 구분된 심층 분석 | 3-5개 섹션 |
| **왜 주목받나?** | 이슈의 사회적/문화적 의미 | 1문단 |
| **앞으로의 전망** | 향후 예상되는 전개 | 1문단 |
| **마무리** | 독자 참여 유도 질문 포함 | 1-2문단 |

## 3. 작성 스타일

### 3.1 문체

- **존댓말 사용**: "~입니다", "~습니다" 체
- **객관적 톤**: 사실 기반 서술, 감정적 표현 자제
- **친근함 유지**: 마무리 섹션에서 독자와 대화하듯 작성

### 3.2 헤드라인 작성법

**좋은 예:**
- "BTS 아리랑, 빌보드 200+핫100 동시 1위 충격! 7번째 정상"
- "국제 유가 100달러 돌파! 기름값 폭등에 RE100 대응 시급한 이유?"

**원칙:**
- 숫자와 구체적 사실 포함
- 느낌표(!) 또는 물음표(?)로 흥미 유발
- 핵심 키워드를 앞쪽에 배치

### 3.3 소제목 작성법

- 질문형 또는 강조형 사용
- 예: "유가 100달러 시대의 재림, 무엇이 가격을 올렸나?"
- 예: "흔들리는 장바구니 물가, 서민 경제 '비상'"

| 카테고리 | 예시 |
|----------|------|
| **연예** | K-POP, 방송, 영화, 아이돌 |
| **경제** | 주식, 환율, 유가, 기업 뉴스 |
| **정치** | 국내 정치, 선거, 정책 |
| **사회** | 사건/사고, 시위, 사회 이슈 |
| **국제** | 해외 뉴스, 국제 관계 |
| **IT/과학** | 기술, AI, 반도체 |

## 5. 태그 작성 가이드

### 5.1 태그 유형

1. **주제 태그**: 핵심 주제 (예: BTS, 국제유가)
2. **인물 태그**: 관련 인물 (예: 이재명, 방탄소년단)
3. **검색어 태그**: SEO용 키워드 (예: 오늘이슈, 실시간검색어)
4. **카테고리 태그**: 분류용 (예: 연예뉴스, 경제뉴스)

### 5.2 태그 수

- 최소: 10개
- 권장: 15개
- 최대: 20개

---

## 6. 이미지 가이드

### 6.1 썸네일 이미지

- **비율**: 1200x630px (소셜 미디어 최적화)
- **형식**: PNG, JPG
- **출처**: 저작권 자유 이미지 또는 출처 명시

### 6.2 본문 이미지

- 주요 이슈 설명에 도움이 되는 이미지 1-2장
- 캡션 포함 권장
- Markdown 형식: `![설명](URL)`
